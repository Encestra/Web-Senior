Sigamos aprendiendo como utilizar WebSenior

Cuando veas esta manita señalando a un elemento significa que puedes clickearlos u oprimirlos para que suceda algo

Cuando veas este elemento llamado "Recuadro" lee y sigue las instrucciones, en estos recuadros podras escribir despues de clickearlos u oprimirlos

Cuando veas este elemento llamado "Boton" clickea u oprimelo, este elemento sivre para realizar una accion, cada boton tendra escrito que accion hace

Cuando veas alguna flecha podras avanzar o regresar a una pantalla, para regresar utiliza la flecha que apunta a la izquierda y para avanzar utiliza la flecha que apunta a la derecha
