Sigamos aprendiendo

Cuando veas una manita señalando, puedes hacer clic con el ratón o tocar con tu dedo si es una pantalla táctil.

Cuando veas este elemento llamado "Recuadro" lee y sigue las instrucciones, en estos recuadros podrás escribir después de hacer clic suavemente en el botón izquierdo del ratón u oprimirlos en la pantalla

Cuando veas este elemento llamado "Botón" seleccionalo con el botón izquierdo del raton u oprimelo, este elemento sivre para realizar una acción, cada botón tendrá escrito que acción se va hacer

Cuando veas alguna flecha podras avanzar o regresar a una pantalla, para regresar utiliza la flecha que apunta a la izquierda y para avanzar utiliza la flecha que apunta a la derecha
